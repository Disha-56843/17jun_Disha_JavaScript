Q:1 What is JQuery?



JQuery is library of JavaScript.
JQuery is a client-side scripting language.
JQuery is light weight.
JQuery is used as programming language.
JQuery is a dynamic computer programming language.
Jquery is platform independent.(Which means it works for all major browsers)
JQuery is case sensitive.
JQuery implementations allow client-side script to interact with the user and make dynamic pages
Can not runs stand alone.(JQuery work with HTML and CSS)
You can save the JQuery file using ".js" extension
JQuery is called through CDN or library.
CDN stands for content delivery network.
JQuery is always fluctuate and called inside <script> tag.

How to start JQuery:

  statement;

</script>
