Q:3 How to create slider with animation?



Syntax:

<script src="https://malsup.github.io/jquery.cycle.all.js"></script>
<script>
  $(document).ready(function () {
    $("Id_name").cycle("Effect");
  });
</script>

Example:



<script>
  $(document).ready(function () {
    $("slider").cycle("turnRight");
  });
</script>
