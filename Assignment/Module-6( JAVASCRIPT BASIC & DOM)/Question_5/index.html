Q:5 What are undeclared and undefined variables?

   Undeclared variavles are those which are not defined or declared. and undefined variables are those which was declared but not given value to them.

Undeclared:

When a variable is “undeclared,” it means that it has not been declared or defined in the current scope.This can happen if you try to use a variable without declaring it first, or if you misspell the variable name.

Example:

  function calc(){
    c = a+b;
  }
  calc(); // ReferenceError: c is not declared

Undefined

"Undefined" variables have been declared in the current scope, but have not been given a value.
Undefined is not a keyword.
If we use 'typeof' operator to get the value of an undeclared variable, we will face the runtime error with the return value as "undefined".
The scope of the undeclared variables is always global.

Example:

  function calc(){
    let c;
    document.write(c);
  }
  calc(); // ReferenceError: c is not defined
