Q:16 How can a page be forced to load another page in JavaScript?



We can use (window.location) property inside the script tag to forcefully load another page in Javascript.
It is a reference to a Location object that is it represents the current location of the document.
We can change the URL of a window by accessing it.

Syntax:


window.location = (Path / URL);

we can change the URL of our window and thus successfully load any page forcibly from our Javascript without any href tag.
