    • Q:4 What do you know about the Bootstrap Grid System?

  • Bootstrap's grid system allows up to 12 columns across the page.
  • Bootstrap's grid system is responsive, and the columns will re-arrange depending on the screen size.
  • If you do not want to use all 12 column individually, you can group the columns together to create wider columns


    • Example:


    one column
    span two columns
    span three columns
    span four columns

    Syntax:


    <div class="row">
      <div class="col-md-1 border border-1">one column</div>
      <div class="col-md-2 border border-1">span two columns</div>
      <div class="col-md-3 border border-1">span three columns</div>
      <div class="col-md-4 border border-1">span four columns</div>
    </div>
