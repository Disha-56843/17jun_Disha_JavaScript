    • Q:2 What is a Bootstrap Container, and how does it work?

  • Containers are most basic layout. These entities surround the elements inside them with a certain amount of padding and margin.
    • Container Types:
    Container
    The div containing the .container class will have a fixed left and right margin and will not occupy the entire viewport or the parent div.
    Container class
    Container-fluid
    The .container-fluid class is used to build a full-width container that spans the entire width of the viewport.
