    • Q:1 What are the advantages of Bootstrap?

  • Bootstrap is an open-source CSS framework designed to come up with mobile-friendly, responsive front-end web development.
    • Advantages:
    Responsive:
    Bootstrap is used as a responsive framework to make it open on all types of devices.
    Support JavaScript and CSS:
    Generally, JavaScript and CSS are used in coding to create a website. Bootstrap supports both these programming languages well
    Customizable:
    It is developer friendly and can be customized as per the project requirements.
    Grids:
    Bootstrap holds 12 column grid styles and supports responsiveness.
