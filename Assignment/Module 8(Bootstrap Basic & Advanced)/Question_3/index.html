    • Q:3 What are the default Bootstrap text settings?

  • Bootstrap 4 uses a default font-size of 16px, and its line-height is 1.5.
  • The default font-family is "Helvetica Neue", Helvetica, Arial, sans-serif.
  • <p> elements have margin-top: 0 and margin-bottom: 1rem (16px by default).
  • Display headings are used to stand out more than normal headings (larger font-size and lighter font-weight), and there are four classes to choose from: .display-1, .display-2, .display-3, .display-4

Bootstrap default settings


Text Color: Bootstrap use the browser's default black as default text color(text-dark).

Heading Styles: Bootstrap provides styles for heading elements (h1 to h6).

Line Height: The default line height is 1.5.

Font Weight: Bootstrap uses a font-weight of 400 for normal text and 700 for bold text.

Text Align: Text is aligned to the left by default.

Text Transform: Bootstrap sets text-transform: none; by default, meaning text is displayed as entered.
