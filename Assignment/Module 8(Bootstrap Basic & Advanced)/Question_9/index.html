    • Q:9 How can one create an alert in Bootstrap?

  • Bootstrap provides an easy way to create predefined alert messages.

  • Alerts are created with the .alert class, followed by one of the four contextual classes .alert-success, .alert-info, .alert-warning or .alert-danger to indicate the type of alert.


    • Example:
    Success!
    Info!
    Warning!
    Danger!
